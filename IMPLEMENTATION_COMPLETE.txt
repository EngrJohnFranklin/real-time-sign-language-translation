═══════════════════════════════════════════════════════════════════════════════
HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════════

FLOW DIAGRAM:
─────────────
Camera detects hand
    ↓
SignRecognizer.process_frame()
    ↓ (returns SignResult with gesture name)
CameraController._on_frame_received()
    ↓ (applies temporal smoothing)
RecognitionService.update()
    ↓ (when stable)
SignController.on_sign_recognized(gesture_name)
    ↓
RealTimeTranslator.update(gesture_name)
    ├─→ Compare: is this a new gesture?
    │   ├─ If same: return (None, False) ✗
    │   └─ If new: get meaning, check confidence
    ├─→ Return (natural_language_meaning, should_speak)
    └─→ Yes means "trigger TTS"
    ↓
UI Update:
├─→ MainWindow.update_translation(meaning)
├─→ SpeechService.speak_async(meaning)
└─→ SignController._schedule_timeout_check()
    ↓
Every 500ms: _check_timeout()
    └─→ If 3+ seconds since last update:
        ├─→ Clear display
        └─→ Stop checking
